### Тема 1: Классы и Объекты - Основные Понятия

**Что такое класс?**
Класс — это чертёж или шаблон для создания объектов. Он определяет:
- **Свойства (Attributes/Fields)** — что характеризует объект
- **Поведение (Methods)** — что объект может делать

**Что такое объект?**
Объект — это конкретный экземпляр класса с конкретными значениями свойств.

**Аналогия из реальной жизни:**
- **Класс** = чертёж дома
- **Объект** = конкретный дом, построенный по этому чертежу

```java
// Определение класса
public class Car {
    // Свойства (fields)
    String brand;
    String model;
    int year;
    
    // Метод
    void displayInfo() {
        System.out.println(brand + " " + model + " (" + year + ")");
    }
}

// Создание объектов (экземпляров класса)
Car car1 = new Car();
car1.brand = "Toyota";
car1.model = "Camry";
car1.year = 2020;

Car car2 = new Car();
car2.brand = "BMW";
car2.model = "X5";
car2.year = 2023;

// car1 и car2 - разные объекты одного класса Car
car1.displayInfo(); // Output: Toyota Camry (2020)
car2.displayInfo(); // Output: BMW X5 (2023)
```

**Важно:** Каждый объект имеет **независимые копии свойств**. Изменение `car1.brand` не влияет на `car2.brand`.
