# Неделя 1: Синтаксис и основы Java
### Что я выучил

1. Переменные и типы данных: int, double, boolean, String, char
   Как объявлять переменные
   Как выполнять операции (+, -, *, /, %)
2. Условия (if/else/switch)
   Операторы сравнения (==, !=, >, <, >=, <=)
   Логические операторы (&&, ||, !)
   switch для множественных вариантов
3. Циклы (for, while, do-while)
   for для известного количества итераций
   while для неизвестного количества
   do-while гарантирует минимум одно выполнение
4. Массивы
   Создание и инициализация
   Доступ по индексу
   Итерация (for и for-each)
   Базовые операции (сумма, максимум, фильтрация)

### Сложные моменты

* Индексы начинаются с 0 (а не с 1)
* Цикл for-each удобнее for с индексом
* Надо не забывать break в switch, иначе fall-through
* Создание нового массива с фильтрацией требует двух шагов: подсчёт +
заполнение
