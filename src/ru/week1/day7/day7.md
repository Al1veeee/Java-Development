## 🎯 Изучено на дне 7

### 1. Разбиение числа по цифрам
```java
String s = String.valueOf(number);     // "321" 
char[] numbers = s.toCharArray();      // ['3','2','1']
int[] digits = new int[numbers.length];

for (int i = 0; i < numbers.length; i++) {
    digits[i] = numbers[i] - '0';      // '1'(49) - '0'(48) = 1
}
```
