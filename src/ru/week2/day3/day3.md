# День 3: Область Видимости Переменных (Scope)

## Уровни видимости:

1. **Локальные переменные** — существуют только внутри метода/блока

```
public static void example() { 
    int localVar = 10; // Виден только в методе example()
}
```

2. **Переменные класса** (статические) - доступ ко всем методам класса 

```
public class Calculator {
    static int globalCounter = 0; // доступна везде

    public static void increment() {
        globalCounter++; // используем глобальную переменную
    }
}
```

3. **Переменные экземпляра** - принадлежит объекту (неделя 3)
