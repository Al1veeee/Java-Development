# Java Development

Репозиторий с моими учебными материалами и практическими заданиями по Java.

## Структура проекта

- `src/ru/week1` — основы Java: переменные, условия, циклы, массивы.
- `src/ru/week2` — методы, параметры, возвращаемые значения, область видимости.
- `src/ru/week3` — задачи и практика за третью неделю.
- `src/ru/week4` — теория и практика, разбитые по задачам.
- `src/ru/week5` — теория и практика пятой недели.
- `java-basics/week1/day1-variables` — ранние заметки и задания по переменным.

## Что есть в репозитории

- конспекты по темам;
- практические задания по дням и неделям;
- примеры структуры учебного Java-проекта.

## Зачем этот репозиторий

Этот репозиторий используется для изучения Java, закрепления базовых тем и хранения учебных материалов по неделям.

## Планы

- привести структуру каталогов к единому стилю;
- добавить больше описаний к неделям и задачам;
- постепенно дополнять решениями и примерами.
